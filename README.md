生活优选（后端）

生活优选是一个基于SpringBoot+Redis的仿大众点评项目

技术栈：SpingBoot+ nginx+ Mysql+ MyBatis-Plus+ Redis

项目亮点：

1.使用Redis解决了在集群模式下的Session共享问题，使用拦截器实现用户的登录校验和权限刷新

2.基于Cache Aside Pattern策略（双写方案）解决数据库与缓存的一致性问题

3.使用Redis对高频访问的信息进行缓存,降低了数据库査询的压力,解决了缓存穿透、雪崩、击穿问题

4.使用 Redis分布式锁解决了秒杀商品在集群模式下一人一单的线程安全问题，使用乐观锁以及sql语句原子性解决超卖问题

5.使用Redis+Lua脚本实现对用户秒杀资格的预检,并基于stream结构作为消息队列,实现异步秒杀下单

6.使用Redis的 ZSet 数据结构实现了点赞排行榜功能,使用Set 集合实现关注、共同关注功能。
